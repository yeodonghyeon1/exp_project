# exp_project


노안 및 시각장애인 식품 탐지 애플리케이션
#폴더 설명

app -> 안드로이드 스튜디오 어플리케이션
data -> flaskserver의 데이터 분석을 통해 나온 데이터 저장소
DeepTextRecogntionBenchmark -> OCR 모델 제작
exam_file -> 실험 example 파일
model -> 훈련된 모델 저장소
test_app -> 임시 테스트 앱
flaskserver -> 파이썬 웹서버

#앱 이용 

안드로이드 스튜디오에서 app 프로젝트 열어서 ip주소 넣는 값에 서버 주소 입력(2곳있음)
flaskserver 서버 실행
